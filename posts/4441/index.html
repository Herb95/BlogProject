部署个人博客方法

发表于 2018-10-21 | 更新于 2018-10-23 | 阅读次数：

部署个人博客命令
npm+hexo

  • npm install hexo-cli -g #安装Hexo

  • npm install hexo-deployer-git –save #部署hexo到git page的deployer

  • hexo init #初始化

  • hexo generate #简化 hexo g 生成
  • hexo server #简化 hexo s 启动服务器

配置github

  • cd ~/.ssh #查看本机是否存在SSH Key
    ###存在
  • clip <~/.ssh/id_rsa.pub #复制公钥到 clipboard
  • 登录github 设置秘钥
  • ssh -T git@github.com #测试秘钥

    不存在

  • ssh-keygen -t rsa -C “your_email@example.com“

    • 弹出界面
    • 直接回车:默认文件进行存储
      • 输入特定的文件名
      • 输入密码(可不输入直接回车)
      • 确认密码
    • 执行存在那一步

  • 创建库

  • 选择设置的SSH 并复制
  • 打开本地hexo项目 编辑 _config.yml

  • 修改

  • 在Hexo 文件下执行

  • hexo g -d
  • 报错 not fount:git
  • 重新安装npm install hexo-deployer-git –save
  • 再次执行 hexo g -d
  • 浏览器输入github生成的外链 https://xxxx.github.io/

美化自己博客

主题

  • 下载网址:https://hexo.io
  • 克隆主题到theme目录

  • 修改配置文件 _config.yml

  • 域名解析

  • 打开https://www.dnspod.cn/ 注册账户
  • 添加域名
  • 添加记录
  • 1.主记录为@
  • 2.www
  • 记录值相同 博客主页对应ip
  • 获取ip ping “github博客地址”
  • 等生效
