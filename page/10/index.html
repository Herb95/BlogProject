CentOS Instal Report

发表于 2024-02-06

目录介绍

  • / 根目录,存放系统命令和用户数据等
  • /boot boot laoder 的静态链接文件,存放与 Linux 启动相关的程序
  • /home 用户目录,存放普通用户的数据
  • /tmp 临时目录
  • /usr 是 red hat linux 系统存放软件的地方,如有可能应讲最大空间分配给它
  • /usr/local 自己安装程序在此
  • /var 不断变化的数据,服务器的一些服务在下面.
  • /opt 附加的应用程序软件包

一般最小需要两个分区,需要一个 SWAP 分区,和一个”/“ 分区,但吧一些常用、重要的挂在点分到其他分区，这样便于管理。一般一个”/“分区,一个 “/usr”分区,一个”/home”分区,一个”/var”分区.

建议系统分区方案

目录大小注解
/swap8G内存小于 4G 的为内存的 2 倍,内存大于 4G 的和内存大小一致
/boot300-800Mcentos8 目录容量要求比较高
/30-50G根目录 较大的一些,强制主分区
/var30-50G存放各种日志文件最好大一些
/usr50-100G用户个人配置应用程序,特别是安装一些容易产生日志较大的程序若 wowza tomcat 等时
/home10G
/opt10G
/tmp10G
